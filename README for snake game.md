## 🛠️ Technologies Used
The entire game is built using pure Python and built-in libraries without any heavy external game engines:
* **`curses`**: For handling the terminal screen, coordinates, and real-time keyboard inputs.
* **`random`**: For generating the food coordinates at random locations on the screen.

---

## 🕹️ How to Play
* Use the **Arrow Keys** (`UP`, `DOWN`, `LEFT`, `RIGHT`) to navigate the snake.
* Eat the food character to grow and increase your score.
* Avoid crashing into the screen borders or into the snake's own body, or it's Game Over!
